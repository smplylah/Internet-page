Internet Protocols and IP Address

Internet Protocols (IP) are rules or standards that allow devices to communicate over the internet. Their main function is to define how data is packaged, addressed, transmitted, and received.

Examples of common internet protocols include: HTTP (HyperText Transfer Protocol), HTTPS (Secure HTTP), FTP (File Transfer Protocol), TCP/IP (Transmission Control Protocol/Internet Protocol), and SMTP (Simple Mail Transfer Protocol).

Types and Versions of IP Addresses

An IP address is a unique string of numbers used to identify devices on a network. It allows devices to send and receive data across the internet.
